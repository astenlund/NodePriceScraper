# NodePriceScraper

Proof of Concept for getting book prices off of online bookstores. Only one website is supported in this initial version, but this can easily be extended. Uses Google's Puppeteer.

# Requirements

Node.js (https://nodejs.org/en/download/)

# Usage

```
npm install
node .\app.js --headless --isbn 9788245014457 9788202562595 9788249518364
```
